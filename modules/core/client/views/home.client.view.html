What is an image board?

An imageboard or image board is a type of Internet forum which operates mostly via posting images. The first imageboards were created in Japan, and inspired the creation of a number of English language imageboards. They are based on the textboard concept.

Imageboards are similar to bulletin board systems, are used for discussions of a variety of topics. Imageboards are also different from online galleries in that most of the works posted are not made by the poster, but instead are taken from other online sources: galleries, other imageboards, and edited pictures.

How to image board?

The basic stuff

  • You don't need to be registered (but you can), you can be anonymous. But we use cookies to assign special IDs to a post, this way you can see if you are talking with the same person or not
  • In the navigation bar (top of the page) you can find the Threads button to list the actual threads on the board.
  • Each thread can have multiple posts discussing a topic.
  • The list of threads is ordered from newest to oldest. In the list the threads are shown 6 prer page, use the pagination at the end of the list of threads

Threads

  • A thread can have a especific topic, but it's optional.
  • To create a thread you need to make a post, the thread post need to have a file attached to them and a comment.
  • The person that makes a thread (and the first post) is called OP (Original Poster).

Posts

A normal post looks like this:

Anonymous (ID:vd8FqqhU) 10/17/18(Wed) 16:48:22 No. 42 >>43 >>55
File: default.png ( 64.2 KB 128 x 128)

This is a coment
This is a reply link to another post in the thread (No. 20):
>>20
This is a reply to the Original poster:
>>30(OP)

This is a reply to a post that I made earlier:
>>33(you)
This is a reply to a post that have been deleted or doesn't exist inside the thread:
>>100

At the top of the post is the information section:

  • The name of the poster. Default is Anonymous. When creating a post there is a field to put a name if you want. If you are signed in the name would be preceded by two # characters (i.e. ##username).
  • The special ID. You would have an unique ID for each thread. This IDs are tracked with cookies.
  • The dete and time when the post was made. The format of the date is Day/month/year.
  • Number of the post. If you click the number you can reply to that post. This numbers are global on the page, that means that a thread doesn't start with No. 1, it continues from the last post in the database.
  • The reply button. If you find hard to clic the number of the post to reply, you can just clic the arrow button and it would work the same.
  • The reply links. This are links to post that have replied to that post, you can click on them and they will scroll to the replies.

The next section is the file information:

  • The name and type of the file. Posts accepts formats: .jpg, .png, .gif and video .webm.
  • The size in B, KB or MB. Posts accepts 5MB or less.
  • The width and height of the file, if it's a video this isn't shown.
  • If there is no file attached the file information section is not shown.

The last section is the content:

  • In the left, the thumbnail of the file. If the file is larget than 125 px in width or height is going to be resized. You can clic on it to enlarge it. Clic again to make it smaller again. Files smaller than 125 px will not be resized nor can be enlarged.
  • If there is no file attached this section is not going to appear.
  • In the right, the comment of the post. A normal post (Not OP) must have either a file or a comment, can have both, can not be empty.
  • A comment can have up to 2000 characters.
  • Comments can have links to other posts inside the thread. This links allways need to have the >> characters to be identified as post links. If you clic on them they will scroll you to that comment.
  • A link that ends with (OP) means that if you clic on it, it will sroll to the first post of the thread.
  • A link that ends with (you) means that it's a reply to one of your post inside the thread.
  • A link that is corssed by a line is an invalid link. It's either a deleted post or that number doesn't exist inside the thread.
  • Editing or deleting posts is not posible, only Administrators can do it. It's the same for threads.
